# Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data using Power BI. The dataset was cleaned and structured in Excel, and an interactive dashboard was created to analyze sales performance, profit trends, and customer behavior.

## Tools Used
- Microsoft Excel (Data Cleaning)
- Power BI (Data Visualization)
- DAX (Basic Measures)

## Key Features
- KPIs for Total Sales, Total Profit, and Quantity
- Monthly and yearly sales trend analysis
- Analysis by customer segment, category, and region
- Interactive slicers for region-wise filtering

## Files in this Repository
- Cleaned Excel dataset
- Power BI (.pbix) dashboard file
- Dashboard screenshot
